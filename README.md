### Clone Project
`NodeJs Version >= 12.0.0`

Clone project
  ```shell

  $ git clone https://github.com/alissonpeixer/spotify-web-clone-react

```

 Go to project folder
```shell

  $ cd spotify-web-clone-react/
```

  Install dependency
  ```shell
  
  $ yarn  or  $ npm
  
  ```
  
  Start local server
  ```shell
  
  $ yarn dev
  
   ```
   
  Start host server
  ```shell
  
  $ yarn dev --host
  
  ```
